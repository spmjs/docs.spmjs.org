spm-help

Table of Contents

  • 安装
  • 使用说明

安装

npm install spm@2.x -g

使用说明

spm help 可显示某个命令的帮助信息，比如想看 install 的帮助信息。

spm help install

也可以使用每个命令的 -h 参数，这两种方式是相同的。

spm install -h
