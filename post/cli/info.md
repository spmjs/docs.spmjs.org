# spm-info

- pubdate: 2013-08-07
- category: 包管理工具
- index: 6

-----------

## 安装

```
npm install spm@2.x -g
```

## 使用说明

info 可查看指定模块的信息。

```
$ spm info arale/dialog
```

可查看整个 family 下的模块信息

```
$ spm info arale // 查看
```

## 选项

### -s, --source

指定源，详情看 [配置文档](../doc/spm-global-config#source)。
